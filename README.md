# 🎓 AcadIQ — Smart Student Analytics System

A full-stack REST API built with Python and Flask that processes student academic results, performs statistical analysis, predicts student performance using Machine Learning, and generates professional reports.

---

## 🚀 Features

- **Student Management** — Add, update and retrieve student records
- **Result Processing** — Add marks per subject, bulk upload via CSV
- **Analytics Engine** — GPA, percentile, rank, grade distribution, class statistics
- **At-Risk Detection** — Automatically flags students likely to fail
- **ML Predictions** — Predicts final score and risk level using Linear Regression + Decision Tree
- **PDF Report Cards** — Auto-generated per student with insights
- **Excel Export** — Multi-sheet analytics workbook with leaderboard and subject averages

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python 3, Flask |
| Database | SQLite |
| Data Processing | Pandas, NumPy, SciPy |
| Machine Learning | Scikit-learn |
| PDF Generation | fpdf2 |
| Excel Export | openpyxl |
| API Testing | curl / REST Client |
